## §3. HTML 산출 체크 (§④)

| 체크 | 기준 | FAIL 시 |
|------|------|--------|
| 단일 HTML | CSS+JS 인라인 | 분리 = 재통합 |
| 모바일 퍼스트 | max-width 430px | 데스크톱 고정 = 재작성 |
| 탭 네비 | 하단 3~5개 | 누락 = 추가 |
| 컬러 | 그레이스케일 + 브랜드 1 | 브랜드 2+ = 재작성 |
| 폰트 | system-ui | 외부 폰트 의존 = 교체 |
| 한글 실데이터 | Lorem/John Doe ✗ | 감지 = 재작성 |
| 필수 5화면 | 온보딩·홈·상세·행동·마이 | 누락 = 추가 |
| 탭 전환 JS | onclick 이벤트 | 정적 = JS 추가 |
